# FruitBoxes-

I have built an API for managing fruit boxes. The main purpose of the project is to display a list of fruit boxes.


All endpoints can be tested using Postman or any similar API client.
The Express server runs on port 8001.

# End points-
The required endpoints are:
* getAll() --> all fruit boxes.
* getById() --> specifik fruit box
* addFruitBox() --> a list of boxes including a specific fruit

# How to get started -
1. Clone the repo xxxx
2. Install the needed dependencies
   npm install
4. run the express server
   npm run dev
